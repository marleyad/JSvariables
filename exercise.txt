 
1. What is the difference between var and let?

  var had a global scope. let has a block scope. var was also the first variable type for Javascript. You are able to redeclare with a var variable but you aren't able to do that with let. var is also 'hoistable'

2. What is the difference between var and const? 

  var has a global scope and const has a block scope. You are able to use var for reassigning and redeclaring. You can't reassign or redeclare with const. 

3. What is the difference between let and const?

  You can reassign let but you can't do that with const. 

4. What is hoisting?

  Variables are carried to the top of the scope in which they are declared in. Using var, you can access the variable name before it is used later on.
